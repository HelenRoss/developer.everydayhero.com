List all Charities

GET /charities

Visibility

Public.

Parameters

campaign_id
Optional string. Retrieve only the charities participating in a campaign.

Response

{
  "charities": [
    {
      "id": "au-1",
      "name": "Starlight",
      "slug": "starlight",
      "country_code": "au",
      "description": null,
      "gift_aid": false,
      "logo_url": "http://"
    }
  ]
}
