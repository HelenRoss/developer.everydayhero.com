Create a Team

  • Must be created from an existing individual page

  • Creates a Team, Team Page and sets individual page as team leader

    POST /teams

Visibility

Restricted. Requires an authenticated user.

Payload

individual_page_id
Required integer. The id of the individual page to create the team from.
name
Optional string. The name of the team. This needs to be unique within the campaign.
{
  "individual_page_id": 1,
  "name": "My New Team"
}
