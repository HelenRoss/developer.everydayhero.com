Campaigns

The Campaigns API allows you to list and view campaigns.

  • List all Campaigns
  • View a single Campaign

List all Campaigns

GET /campaigns

Visibility

Public.

Parameters

charity_id
Optional string. Retrieve only the campaigns that the specified charity is participating in.

Response
